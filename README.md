# Agent Library (Plane-Targeted)

Unzip and use this folder as your complete library.

## How to add a doc
1) Start from `templates/DOC_PLANE_HEADER_TEMPLATE.md` (or a plane template).
2) Fill in the header fields.
3) Save under `planes/<TargetPlane>/<category>/`.

## Runtime enforcement
Placement and linting rules live under:
- `planes/governance/rules/DOC_HEADER_LINTER_RULES.md`
- `planes/governance/rules/LIBRARY_PLANE_PLACEMENT_RULES.md`
